# 🪄 Hogwarts – Aplicación Android

Este proyecto fue desarrollado como parte de la materia **Aplicaciones Móviles**, utilizando **Android Studio**, con el objetivo de crear una aplicación móvil funcional inspirada en el universo de *Harry Potter*.

La propuesta consiste en una mini plataforma que simula un campus de Hogwarts, donde los usuarios pueden registrarse, iniciar sesión y gestionar información dentro de la aplicación.

---

## 🎯 Objetivo del proyecto

Desarrollar una aplicación Android aplicando conceptos de desarrollo mobile, incorporando:

- Gestión de usuarios.
- Autenticación mediante Firebase.
- Persistencia de datos.
- Operaciones CRUD.
- Validación de entradas.
- Manejo eficiente de procesos mediante hilos secundarios.

---

## ✨ Características

- Registro de nuevos usuarios.
- Inicio de sesión mediante Google.
- Autenticación utilizando Firebase.
- Validación de datos para evitar registros incorrectos o repetidos.
- CRUD de usuarios.
- CRUD de entidades propias del sistema.
- Base de datos utilizando Firebase.
- Interfaz inspirada en un campus mágico de Hogwarts.
- Uso de recursos externos para textos mediante `strings.xml`.
- Implementación de tareas complejas mediante hilos secundarios para mejorar la experiencia de usuario.

---

## 🏰 Temática de la aplicación

La aplicación recrea una experiencia similar a un campus virtual de Hogwarts, permitiendo a los usuarios interactuar con distintas funcionalidades dentro de un entorno inspirado en la academia de magia.

---

## 🧠 Tecnologías utilizadas

- Android Studio
- Java 
- Firebase Authentication
- Firebase Realtime Database / Firestore
- XML (Layouts)
- Android SDK

---

### ⚡ Experiencia de usuario

Para evitar bloqueos en la interfaz, las operaciones que requieren mayor procesamiento son ejecutadas utilizando hilos secundarios, manteniendo una navegación fluida dentro de la aplicación.
